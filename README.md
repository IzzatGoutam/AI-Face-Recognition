# AI Face Recognition System

A real-time AI Face Recognition project built using Python, OpenCV, and the face_recognition library.

This system detects and recognizes faces through a webcam and displays the person's name on the screen.

---

## Features

- Real-time face detection
- AI-based face recognition
- Webcam integration
- Displays detected person's name
- Multiple face image support
- Simple and beginner-friendly project

---

## Technologies Used

- Python 3.11
- OpenCV
- face_recognition
- dlib
- NumPy

How It Works
The system loads images from the dataset folder.
Face encodings are generated using AI.
Webcam captures live video.
Detected faces are compared with stored encodings.
The recognized person's name is displayed on screen.
Output
Green rectangle around detected face
Name displayed above the face
Live webcam recognition
